## Summary
**﻿Sản phẩm nhóm - Ứng dụng mô phỏng Mobile Banking** được xây dựng theo kiến trúc **Client-FirebaseCloud Functions** có các chức năng cơ bản:

- Quản lý Người dùng
- Quản lý Giao dịch
- Quản lý Tài khoản
- Đặt vé xe / vé phim
- Định vị

**Công nghệ:** Java - Android Studio - Gradle.
